Social Stars is a monorepo Next.js application deployed on Vercel. All server logic lives in API routes inside the nextjs_space/app/api/ directory. The database is PostgreSQL accessed via Prisma ORM. Authentication uses NextAuth with a JWT strategy — the user's id and role are embedded in the token. The AI activity generator calls Anthropic's Claude Opus 4.6 model server-side.

Role-Based Access Control

RolePages & RoutesRestrictions
Public /  /get-started  /auth/*  /enroll  /api/activities  /api/signup No session required
Parent /select-child  /dashboard/[id]  /activity/*  /parent-dashboard role = "parent"; childId passed via URL query param
Therapist /therapist  /therapist/enroll  /therapist/generate  /story-builder  /api/therapist/* role = "therapist"; TherapistFamily link verified on every data mutation

Role is embedded in the NextAuth JWT token on login via a custom jwt() callback and forwarded to session.user.role. All API routes re-fetch the DB record to verify role — they never trust the token alone for write operations.

Key Technical Patterns

🔐
JWT Auth
NextAuth with credentials provider. user.id and user.role embedded in JWT. No DB sessions needed.
📦
Prisma Singleton
lib/db.ts exports a single PrismaClient instance to prevent connection pool exhaustion in serverless.
🔊
Web Speech API
speechSynthesis used for audio feedback. No external audio files. Breathing pages cancel speech before speaking.
🎨
Tailwind + shadcn/ui
Radix UI primitives + Tailwind. Child-facing cards use .child-card and .child-button CSS classes.
🎊
Confetti
canvas-confetti fired on correct answers to reward children immediately with visual feedback.
📊
Recharts
Bar and pie charts in parent dashboard showing per-child progress across all activity types.
🤖
Claude Opus 4.6
Adaptive thinking enabled. Generates structured JSON activities server-side. API key never exposed to browser.
🌬️
Breathing Animation
CSS transform: scale() with transition duration matching phase.duration. Expand/contract driven by phase state.
🔥
Streak Calculation
Server-side in progress/route.ts. Counts consecutive calendar days with activity. Resets after 2 idle days.
